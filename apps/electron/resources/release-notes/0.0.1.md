# v0.0.1 — Welcome to CrabPal 🦀

The first public release of **CrabPal** — a desktop AI assistant that connects your data, tools and workflows in one place.

## What's new

### 📥 Seamless import
Bring over your existing agent history with a single click. Sessions, skills, sources and preferences are migrated automatically so you can pick up right where you left off.

### 📁 Projects
Group related sessions, sources and skills under a **Project**. Each project keeps its own working directory, context files and scope — perfect for separating clients, codebases or research threads without cross-talk.

### 👤 Account & Usage
A dedicated view for your account and model usage. Track token consumption, cost and rate-limit status over time, and see which sessions or models are driving your spend.

### 🔌 In-session connection switching
Change the active model or connection (Anthropic / Bedrock / custom) **directly inside a session** without leaving the chat. Swap providers mid-task, compare outputs, or fall back to another backend when one is unavailable.

### ⚡ Rebuilt core
- **Automations** — event-driven workflows that run on schedules, labels or status changes.
- **Skills** — reusable instruction sets shared across Claude Code, Cursor and CrabPal.
- **Sources** — unified MCP / REST / local adapters with OAuth, credentials and soft-validated templates.
- **Permission modes** — Explore (read-only), Ask to Edit, and Execute, switchable mid-conversation.

### 🎨 Polish
- Redesigned sidebar with pinned **What's New** and Settings.
- Improved diff, datatable, spreadsheet, PDF and image previews.
- New theming system with per-workspace colours.

Thanks for trying CrabPal — feedback is very welcome.
