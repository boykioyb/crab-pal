# v0.0.5 — Upstream fixes, smoother agents 🔧

Under-the-hood polish. Long runs are quieter, OAuth stops asking you to re-login mid-session, and a batch of upstream core bug fixes landed.

## What's new

### 🔐 OAuth tokens auto-refresh before each SDK spawn
No more mid-session "please re-login" interruptions when access tokens expire. Every new agent subprocess now picks up the freshest token from the credential store.

### 🧠 `xhigh` thinking level
An extra step up from `high` for problems that warrant more reasoning budget. Available wherever you pick a thinking level.

### 💬 Follow-up quotes sent in full
Replying with a quoted earlier message no longer truncates the quote — the full context reaches the agent. (#580)

### 🛟 `call_llm` recovers partial output
When a sub-agent hits its turn limit, you now get whatever it produced instead of a hard failure. Graceful degrade over silent loss. (#544)

### 🪛 pi-agent stderr surfaced on connection-test failure
Connection-test errors from the Pi backend now show the subprocess stderr, so a misconfigured API key isn't a silent timeout anymore.

## Under the hood
- Ported v0.8.10 upstream core bug fixes (#566, #567, #575, #576).
- Branding centralized; hosted URLs switched to GitHub.
