# v0.0.3 — Updates now ride on GitHub 🚀

A plumbing release. Auto-updates are moving to a new home.

## What's new

### 🔄 Auto-update via GitHub Releases
Check-for-Updates now fetches new versions directly from this repo's **GitHub Releases** instead of a custom domain. No behavior change on your end — just a more reliable delivery path.

### 🔧 Under the hood
- `electron-builder` `publish` provider switched from `generic` to `github`.
- Release workflow now uploads the `latest.yml` / `latest-linux.yml` manifests alongside the Windows and Linux artifacts, so all three platforms can auto-update.
